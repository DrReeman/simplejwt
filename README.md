# SimpleJWT

SimpleJWT is a simple JSON web token library written in PHP.

## Features


## Requirements

- PHP 5.3.9 or later
- `hash` extension
- `openssl` extension

## Installation

You can install via [Composer](http://getcomposer.org/).

```json
{
    "require": {
        "kelvinmo/simplejwt": "dev-master"
    }
}
```
